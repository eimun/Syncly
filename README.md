# 🚀 Syncly – Advanced Task Management System

A high-performance, zero-dependency task management application designed with scalable frontend architecture and efficient state handling.

---

## ✨ Features

* ⚡ Real-time task updates
* 🔁 Undo / Redo functionality (stack-based system)
* 🧠 Pub-Sub architecture for state management
* 🚀 Optimized performance with debounce

---

## 🛠 Tech Stack

* JavaScript (ES6+)
* HTML5 & CSS3
* LocalStorage
